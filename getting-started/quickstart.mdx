Now that you've got Pipecat installed, let's run your first example application. We'll use the "Say One Thing" example, which creates a simple bot that greets users when they join a room.

## Getting the Example

Download the ["Say One Thing"](https://github.com/pipecat-ai/pipecat/raw/main/examples/foundational/01-say-one-thing.py) example from the `pipecat` repo and save it to your project directory.

## Configuring the Example

Depending on which text-to-speech service you're using, you'll need to make some small adjustments to the example:

<AccordionGroup>
    <Accordion title="ElevenLabs">
        1. Import the ElevenLabs module:
        ```python
        from pipecat.services.elevenlabs import ElevenLabsTTSService
        ```

        2. Add this import:
        ```python
        from pipecat.frames.frames import LLMFullResponseEndFrame
        ```

        3. Configure the TTS service:
        ```python
        tts = ElevenLabsTTSService(
            api_key=os.getenv("ELEVENLABS_API_KEY"),
            voice_id=os.getenv("ELEVENLABS_VOICE_ID"),
        )
        ```

        4. After the greeting line:
        ```python
        await task.queue_frame(TextFrame(f"Hello there, {participant_name}!"))
        await task.queue_frame(LLMFullResponseEndFrame())
        ```
    </Accordion>

## Running the Example

1. Open your Daily room URL in a browser tab
2. Join the room
3. Run the example:

```bash
python 01-say-one-thing.py
```

If all goes well, you should see another participant join the room, greet you by name, and disconnect.

## Understanding How It Works

Let's look at the key components of this example:

```python
# Set up the transport (handles audio I/O)
transport = DailyTransport(
    room_url,
    token,
    "Bot",
    DailyParams(
        audio_in_enabled=True,
        audio_out_enabled=True,
    )
)

# Set up text-to-speech service
tts = ElevenLabsTTSService(
    api_key=os.getenv("ELEVENLABS_API_KEY"),
    voice_id=os.getenv("ELEVENLABS_VOICE_ID"),
)

# Create the pipeline
pipeline = Pipeline([
    transport.input(),     # Listen for audio
    tts,                  # Convert text to speech
    transport.output()    # Output the audio
])

# Handle when someone joins
@transport.event_handler("on_first_participant_joined")
async def on_first_participant_joined(transport, participant):
    participant_name = participant["user_name"] or "there"
    await task.queue_frame(TextFrame(f"Hello {participant_name}!"))
```

This example demonstrates several core Pipecat concepts:

- Using a transport for real-time audio
- Setting up AI services
- Creating a processing pipeline
- Handling events and generating responses
